### 个人介绍
出生年月：198904 现居：上海市松江，本科毕业于河南理工大学，专业机械自动及自动化，硕士就读于上海大学，专业是机械工程，研究方向是工业机器人视觉检测及测量。
曾在上海微电子装备有限公司从事开发手机屏幕检测类的项目，现在一家传感器公司从事指纹图像预处理及指纹识别软件及算法的开发，熟悉基础图象操作，在图像预处理，2d图像特征提取，图像匹配方面比较擅长。
